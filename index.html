How to Install Mr Invisible fork

Current Repository Version: 0.2.3

Instructions for adding this repo:

  • Go to the Kodi file manager
  • Click on "Add source"
  • The path for the source is https://Mr-invisibles.github.io/repo/
  • Name it "Mr invisible" without the quotes
  • Head to " My Addons"
  • Select install an addon from zipfile.
  • When it asks for the location, select "Mr invisible", and install repository.mrinvisible-0.2.3.zip.
  • Go back to My Addons and select "Install from repository"
  • Select the "Mr invisible" menu item
  • Open "Video add-ons" and install (Mr Invisible) or the addon you like to use
  • Adult Password = pervx

Enjoy!
